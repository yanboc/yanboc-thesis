# agent

本目录记录与 AI 协作相关的流程、偏好与任务追踪信息，用于模板改造的过程管理。

## 文件总览

- `to_template.md`：模板化改造任务清单与阶段进度。
- `preference.md`：当前协作偏好（语言、输出风格等）。
- `agent.md`：对代理行为的补充说明。
- `latex.md`：LaTeX 写作或编译相关的协作提示。

### 模板改造任务（`to_template.md`）

主要内容：
- 以编号列表记录改造步骤与完成状态。
- 面向“论文转模板”场景，追踪去个性化与结构化工作。

可改项建议：
- 每步增加“输入/输出文件”和“验收标准”。

### 协作偏好（`preference.md`）

主要内容：
- 记录用户在语言、风格、操作顺序方面的固定偏好。

可改项建议：
- 增加“必须执行项/禁止项”两段，便于长期复用。

### 代理说明（`agent.md`、`latex.md`）

主要内容：
- 记录 AI 在本项目中执行修改时的规范与边界。

可改项建议：
- 当编译链、目录结构或约束变化时同步更新，避免“规则漂移”。
