# appendix

本目录用于放置附录内容，承载正文之外但对复现、完整性和可读性重要的材料。

## 文件总览

- `notations.tex`：术语索引页，输出 glossary。
- `experiments.tex`：补充实验细节与完整结果表格（若项目需要）。

### 术语索引（`notations.tex`）

主要内容：
- 定义“术语索引表”章节并调用 `\printglossary` 输出术语。
- 通过局部命令避免强制分页，控制附录版式衔接。

可改项建议：
- 可调整标题显示方式与索引排序/编号呈现。

注意事项：
- 术语条目来源于 `includefile/gls_cmd.tex` 与 `settings/gls.tex`，需联动维护。

### 实验补充（`experiments.tex`）

主要内容：
- 汇总正文未完全展开的实验表格、图像与统计口径说明。
- 包含多数据集、多模型配置下的对比结果与原始数据。
- 可通过章节化组织实验设置、指标定义与补充结果，支撑正文结论。

可改项建议：
- 推荐按“复现实验最小单元”（任务-设置-指标-结果）组织子节。

注意事项：
- 模板化时保留结构与表格接口，移除具体课题结论与专有数据。
