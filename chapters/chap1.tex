\chapter{绪论}

近十年来，基于神经网络（neural network）的深度学习（deep learning）技术凭借其卓越的任务性能与广泛的应用场景，已逐渐成为人工智能领域的支柱性技术，在计算机视觉、自然语言处理、智能推荐等领域实现了突破性进展。
然而，神经网络的 “黑盒” 特性始终制约着该技术的进一步发展与可信应用。
深度学习模型内部的特征（feature）提取与作用机制缺乏合理的可解释性，导致其在安全攸关场景中的部署面临挑战，同时也使得深度学习的发展长期依赖经验驱动，缺乏先验的理论指导。

传统学习理论（learning theory）囿于统计学、最优化理论等基础理论的分析框架，难以适配深度学习模型的高维性、非线性与动态演化特性，与深度学习的实际应用之间存在显著差异。
为弥补这一差异，一系列现代机器学习理论应运而生；
本文所关注的\textit{特征学习理论}\!
\footnote{在本文中，FLT特指一类新兴的深度学习理论分支，而非广义上所有与特征相关研究的统称。为避免概念歧义，我们将在\S\ref{sec-related-works-learning-paradigms}中明确给出FLT的核心界定，并系统对比FLT与其他相关理论体系的异同。}\!\!
（\textit{Feature Learning Theory，FLT}）\cite{CCB2022Benign,AL2021Feature,AL2023Towards,CCL2025Prevalence}便是其中的倍受关注的研究方向之一。
该理论致力于揭示神经网络从数据中提取并利用特征的内在规律，为深度学习的可解释性与可信性提供理论依据。

本章将首先阐述FLT的研究背景与意义，然后系统梳理国内外研究现状，并明确本研究的主要目标与贡献，最后介绍论文的整体结构安排。

\section{研究背景与意义}

FLT作为深度学习领域的新兴基础理论，聚焦于揭示神经网络在学习算法的驱动下从训练数据中自动提取并利用有效特征的内在规律，其研究不仅具有深刻的学术理论价值，更对人工智能高效、安全的落地实践具有关键的指导意义。

相较于支持向量机、逻辑回归等传统机器学习模型，以卷积神经网络（Convolutional Neural Network, CNN）、自编码器（Autoencoder, AE）、Transformer（TF）等为代表的神经网络深度学习模型，具备从图像、语音、视频等高维复杂模态数据中自动提取有效特征的核心能力，从而在图像分类、自然语言处理等复杂任务中实现了性能的跨越式提升。
然而，人们对诸如“神经网络如何自动从训练数据中提取有效特征”、“网络提取的特征如何随训练过程演化”、“为何特定结构的网络只能在一部分学习任务中表现出色”等核心问题缺乏系统性认知，这导致深度学习模型常被视为不可解释的黑盒子，严重阻碍了其理论发展与可信应用。
特征学习理论通过数学建模、理论推导与实证分析，清晰刻画了模型结构、参数初始化与梯度下降对特征学习过程的影响，揭示了诸多深度学习模型特有属性的内在机理（如良性过拟合\cite{CCB2022Benign}、特征纯化（Feature Purification）\cite{AL2021Feature}、神经坍缩（Neural Collapse）\cite{CCL2025Prevalence}等），填补了传统理论与深度实践之间的鸿沟，为深度学习的可解释性提供了有力支撑。

此外，特征学习理论还为不同领域的机器学习问题提供了统一的理论分析框架。
在传统学习理论中，不同领域、场景下问题的分析框架一般也各自独立，所得结论无法相互参考比对。
特征学习理论的研究，能够提炼不同领域场景、不同数据与模型假设下特征提取、优化、迭代的共性规律，构建统一的理论分析框架，打破各研究方向的壁垒。
例如，特征学习理论可以在统一理论框架下分析并揭示知识蒸馏\cite{AL2023Towards}、自监督对比学习\cite{WL2021Toward}与图神经网络学习\cite{HCW2024Graph}等多个不同场景下的前沿问题的底层原理，实现理论认知在计算机视觉、自然语言处理、智能推荐等不同领域的跨场景迁移，推动机器学习理论体系的进一步系统化与完善化。

在实践中，特征学习理论可以指导深度学习模型的设计与优化，提升模型应用的鲁棒性与泛化能力，解决实际场景中的核心技术痛点。
传统深度学习模型设计多依赖研究者的经验与大量试错，具有封闭性与随机性的特点。
特征学习理论的研究成果，能够为模型结构设计、训练策略选择提供明确的理论指导。
例如，基于“SGD 先挖掘后优化特征”的规律\cite{LWY2024Benign}，可针对性优化模型训练的学习率调度策略；基于“特征纯化提升鲁棒性\cite{AL2021Feature}” 的机理，可改进对抗训练算法以增强模型的抗干扰能力；基于“混合增强（Mixup）对特征学习的增益机制\cite{ZCL2023Benefits}”，可设计更高效的数据增强方案。
这些理论指导能够大幅减少模型研发的盲目性，降低试错成本，适用于包括计算机视觉、自然语言处理在内的大多核心任务。
在深度模型的实际应用场景中，数据往往存在分布偏移、噪声干扰、样本稀缺等问题，对深度学习模型的泛化能力与鲁棒性提出了较高的需求。
特征学习理论针对分布外泛化\cite{CHZ2023Towards}（Out-of-Distribution Generalization）、异常检验\cite{CL2026Two-Layer}、Zero-Shot学习\cite{CDL2024Understanding}等场景的研究，能够帮助模型开发者从根本上理解的特征学习过程，进而提升模型的鲁棒性与泛化能力。
在上述示例中，特征学习理论通过挖掘特征与真实任务标签的内在关联，抑制虚假特征的干扰；
通过学习可迁移的通用特征，提升模型在未知数据分布下的泛化性能；通过特征多样化学习，缓解小样本场景下的过拟合问题。
上述分析推动了AI模型在医疗影像诊断、工业质检、智能驾驶、金融风控等安全攸关领域的规模化落地，解决实际场景中的核心技术痛点。

\section{国内外研究现状}\label{sec-related-works}

特征，有时也被称作表示、表征（representation），广义上泛指数据在特定任务空间内的低维刻画，是一个跨越传统机器学习、深度学习与大模型时代的关键概念。
与之对应，广义层面的特征学习理论，其范畴涵盖所有聚焦于特征学习过程的相关研究，其研究范式与核心关注问题，亦随技术与时代的演进发生了显著转变。
本节将从横向（对比FLT与不同研究范式）与纵向（追踪FLT近些年的发展）两个维度，系统梳理国内外关于特征学习理论的研究现状。
由于相关研究发展较快，成果极其丰富，本文仅选择部分代表性成果进行讨论。

\subsection{学习理论的范式演变}\label{sec-related-works-learning-paradigms}

% 神经网络具备可以自动从数据中提取有效特征的能力，却也因此被视为 “黑盒” ；
% 关于神经网络特征学习机制的理论解释始终是学界研究的热点。
早期特征学习理论多基于统计学习理论（statistical learning theory）框架。
该框架数理基础坚实、分析过程严谨，衍生出诸多形式优美的理论成果。
Bengio等\cite{BCV2013Representation}的综述文章系统总结了包括概率PCA（Probabilistic Principal Component Analysis，PPCA）、受限玻尔兹曼机（Restricted Boltzmann Machine，RBM）、自编码器（Auto Encoders，AE）等模型的特征学习原理。
然而，早期特征学习理论始终受限于统计学习理论范式，研究重心集中于经验风险最小化（Empirical Risk Minimization, ERM）与归纳偏置（Inductive Bias）约束两大宏观优化目标，而对数据与模型间的匹配关系、神经元的微观动力学行为缺乏足够关注。
因此，早期特征学习理论无法分析神经网络的精细结构与特性，难以解释实践中所呈现的一系列神经网络特有现象。

随着深度学习技术的蓬勃发展，许多新型学习理论框架相继涌现，一些传统的理论框架也取得了若干新成果。
例如，神经正切核（Neural Tangent Kernel, NTK）通过网络输出对参数梯度的内积定义核函数，在无限宽极限下将神经网络训练动态线性化为核回归问题，为超宽网络的收敛性与泛化性分析提供解析工具；
平均场理论（mean field theory）以群体平均行为近似大规模系统中个体的复杂交互，将高维动力学降维为可解的自洽方程，适用于刻画神经网络特征学习动态或多智能体交互机制；
梯度流（gradient flow）作为梯度下降的连续时间极限，通过常微分方程描述参数演化轨迹，是分析非凸损失曲面几何性质与优化收敛行为的核心框架；
而作为传统学习理论的代表，概率近似正确（Probably Approximately Correct，PAC）理论也被应用到了神经网络上。
该理论通过样本复杂度与置信度量化模型泛化能力，为算法的近似正确性提供概率保证，在深度学习时代也取得了若干新的研究成果。
我们在图\ref{fig-learning-theory-taxonomy}中梳理了当前机器学习领域若干主流的学习理论框架，并附列了各框架对应的代表性研究成果。

2021年前后，Allen-Zhu与Li\cite{AL2021Feature,AL2023Towards}提出了现代特征学习理论研究范式，并引发了学界的广泛关注与讨论。
为避免歧义，本文将使用“现代特征学习”来特指这类新型研究范式，而用“早期特征学习”或“传统特征学习”来指代早期基于统计学习理论的研究。
特别地，英文缩写“FLT”仅用于指代现代特征学习理论。
鉴于FLT仍处于新兴发展阶段，学术界对其核心定义尚未形成统一共识。
为保证研究的严谨性，我们特对 FLT 作出如下界定。
\begin{mynotice}
  \textit{本文所讨论的特征学习理论是一类有限、离散，且算法相关的理论分析框架。
  该理论
  能够分析具体学习任务（由数据建模、模型结构、学习算法等要素共同决定）中模型参数从随机初始化到趋近于收敛的整个迭代过程。}
\end{mynotice}
\noindent 我们对 FLT 的三大核心特性解释如下：
\begin{itemize}
  \item \textbf{有限性}： 以有限宽度网络为研究对象，并不要求网络宽度趋向无穷。相比NTK等无限维理论分析框架，FLT可以得到关于网络宽度的非渐进（non-asymptomatic）的结论，帮助研究者更深入地理解网络参数如何定量地影响网络的表现。
  \item \textbf{离散性}：聚焦于模型参数的离散迭代优化过程，而非将其近似为连续的优化流。因此，FLT不同与梯度流、均值场理论等连续化分析框架，往往不许需要对优化目标做过于理想化的假设。
  \item \textbf{算法相关性}：考虑具体学习算法对特征学习过程的影响，这与 PAC 学习、VC 维理论、表示性理论等传统学习理论的核心假设存在本质区别。
\end{itemize}

\subsection{现代特征学习理论的研究进展}

FLT是一套面向实践应用的理论体系；凡是在实际训练、推理过程中神经网络表现出的性质、现象，都可作为FLT的分析对象与研究载体。
在 FLT 发展初期，研究者主要关注通用学习任务中神经网络的共有性质，例如良性过拟合的发生机制\cite{CCB2022Benign}、随机梯度下降（Stochastic Gradient Descent, SGD）算法的优化动力学机理\cite{LWY2024Benign}，以及对比学习范式下特征表征的学习机制与优化策略\cite{WL2021Toward}等。在本文中，我们针对对抗训练场景下的特征单纯形压缩现象（complex compression phenomenon）展开了系统性分析。我们证明单纯形压缩是神经塌缩（Neural Collapse, NC）现象在对抗训练下的自然推广与延伸。

随着与 FLT 相关探索的不断深入，关于通用任务下神经网络的共性性质研究已日趋完善。
在此背景下，研究者的关注焦点开始转向更为细分、具体的实践场景与技术环节，例如知识蒸馏的特征迁移机制优化\cite{AL2023Towards}、Mixup数据增强策略下的特征分布演化规律\cite{ZCL2023Benefits}等——这类研究均旨在从细粒度层面挖掘特征学习的内在机理，为特定技术方案的优化提供理论支撑。
本文第三章聚焦重构式异常检测（Reconstruction-Based Anomaly Detection, RBAD）场景下的特征学习动力学过程展开系统分析，最终证实：仅在正常数据上完成训练的生成式网络，在对异常样本进行重构时会产生显著更大的误差，基于这一特性可实现对异常点的有效检测。

再然后，随着大语言模型（Large Language Model）时代到来，一些关于语言模型（比如TF）与扩散模型的分析也出现了。本文基于神经网络研究了QA问题中的知识提取。

\section{论文结构安排}

本文共有六章，其中第一章为绪论，介绍了特征学习理论的研究背景与意义，总结了该领域国内外发展近况，并概括了本文的研究内容与主要贡献。
在第二章中，我们结合传统学习理论与FLT相关参考文献，总结了FLT的研究范式，介绍了该领域的基础理论知识。
在第三到第五章，我们分别从通用任务、图像任务与文本任务的角度探索模型的特征学习特性。其中：
\begin{itemize}
  \item 第三章以对抗训练为研究场景，分析全连接神经网络特征学习动力学基础与神经塌缩特性，探究单纯形压缩现象的表征、成因及其与神经塌缩的关联差异，验证其对模型鲁棒性的影响。
  \item 第四章聚焦RBAD场景，探究图像数据与RBAD的适配性，构建卷积神经网络特征学习动力学模型，推导特征提取与数据重建的关联机制并开展实验验证。
  \item 第五章针对问答任务，构建Transformers知识提取动力学模型，解析自注意力机制驱动的知识捕捉与表达逻辑并验证其有效性。第六章对全文研究成果进行总结，分析研究局限性，并展望未来研究方向。
\end{itemize}
本文第三到五章的分析始终围绕着我们在\S\ref{sec-research-questions}中提到的三个研究问题，即“网络如何自动提取有效特征”、“特征如何随训练过程演化”与“为何特定结构的网络在特定类型的数据上具有更优特征学习能力”。
在第六章，我们总结了全文的核心工作，为上述研究问题提供了一份详细的分析、回答。并就未来的潜在研究方向进行了深入思考。
