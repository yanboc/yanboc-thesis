# refs

本目录负责参考文献数据源与文献清理脚本，是引用系统的数据入口。

## 文件总览

- `ref.bib`：BibTeX 文献库。
- `prepare_ref.py`：文献去重脚本。

### 文献数据库（`ref.bib`）

主要内容：
- 维护全文引用所需的条目（会议、期刊、书籍等）。

可改项建议：
- 统一 key 命名（建议“作者-年份-关键词”），减少冲突与歧义。

注意事项：
- 与 `settings/bib_style.tex` 的样式设置联动，提交前应全量检查引用输出。

### 文献整理脚本（`prepare_ref.py`）

主要内容：
- 读取 `ref.bib`，按条目 ID 去重并回写。
- 使用 `bibtexparser` 解析 BibTeX 数据。

可改项建议：
- 可增加“按 DOI/标题近似去重”逻辑，提高鲁棒性。

注意事项：
- 当前脚本按 `ID` 唯一判重；不同 ID 但同文献不会被识别为重复。
